# oa-TOF 当前项目状态

本文件是oa-TOF项目的人类可读权威源。机器参数以
[`../config/baseline.json`](../config/baseline.json)为准；COMSOL、SIMION和CAD文档只能记录
各自实现，不能在本文件之外另行定义项目参数。

## 当前统一定义

- 方案：正交加速飞行时间质谱，双级环栈反射镜，一级10环、二级5环。
- 标准离子：524 amu、+1电荷。
- 初始能量：均值5 eV、标准差0.4 eV，负能量重采样。
- 释放体：`1×1×1 mm³`，当前轴向范围`z=1...2 mm`。
- 分辨率：`R=m/FWHM_m`。窄峰时间域等价式为`R=T/(2*FWHM_t)`。
- `2.3548×sigma`只在高斯假设成立时作为FWHM代理；非高斯峰必须直接测量半高宽。
- SIMION常规统计N=5000；跨求解器严格比较优先使用同一固定ION粒子表。

## 当前几何状态

紧凑加速器稳定候选保持轴向排布不变：孔半宽5 mm、环单边宽5 mm、带电电极到接地屏蔽
每侧5 mm、repeller后表面到接地后盖前表面5 mm、屏蔽壁厚4 mm；物理范围约
`38×38×30 mm³`。SIMION日常网格为`xy=0.25 mm、z=0.05 mm`，收敛参考为
`xy=0.25 mm、z=0.025 mm`。

该紧凑几何尚未写入正式COMSOL MPH和SolidWorks装配体，因此仍是候选，不能称为正式模型。

SIMION检测器第4实例是半径40 mm、有效面`z=19.83 mm`的GUI可见数值终止层，不是机械
检测器实体。当前网格为`dx/dy/dz=0.5/0.5/0.01 mm`，终止层厚0.05 mm，前/背真空余量
0.2/0.05 mm。COMSOL/CAD机械检测器厚度仍按独立机械契约管理。

## 已闭合的SIMION结论

- 524 amu GUI Fly2、N=5000：5000/5000命中。
- 平均TOF约`71.9902651204 us`。
- 直接时间FWHM约`1.392926 ns`，对应`R=25841.38`。
- 直接质量FWHM约`0.02027719 amu`，对应`R=25841.84`。
- 时间域和质量域直接FWHM结果只差约0.0018%。
- 原生splat与`z=19.83 mm`参考面的最大逐粒子时间差约`4.0e-7 ns`。
- 峰为单个非高斯右肩峰；主要来源是初始z到TOF的二次映射，不是漏检、横向截断或绘图伪影。

## COMSOL边界

当前正式COMSOL MPH仍是100 amu历史几何。按统一FWHM口径，其N=1000历史结果为1000/1000
命中、平均TOF`31.4478763926 us`、sigma`0.897802250264 ns`、FWHM`2.1141627354 ns`、
`R=7437.4304`。该结果不能标成524 amu，也不能代表紧凑加速器候选。

524 amu、紧凑加速器、同源粒子表的全三维COMSOL闭合尚待完成。

## 正式化门禁

候选几何只有同时满足以下条件才能转正：

1. COMSOL与SIMION共享几何契约通过。
2. 使用同源粒子比较命中率、平均TOF、直接质量FWHM、分辨率和峰形。
3. 差异有网格收敛或明确物理原因支持。
4. 正式COMSOL MPH已更新并能在GUI中重算。
5. SolidWorks零件和装配体在同一任务更新，保存错误/警告和坐标检查均通过。
