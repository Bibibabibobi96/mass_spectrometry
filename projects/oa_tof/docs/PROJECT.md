# oa-TOF 当前项目状态

本文件是oa-TOF项目的人类可读权威源。人工设计以
[`../config/baseline.json`](../config/baseline.json)为准，程序统一读取自动生成的
`../config/resolved_geometry.json`并按`../config/modes/`运行；COMSOL、SIMION和CAD文档只能记录
各自实现，不能在本文件之外另行定义项目参数。

## 当前统一定义

- 2026-07-18 已验证并转正`baseline → resolved → 自包含SIMION正式包`构建链：正式网格PA、
  四实例IOB、同名Lua/Fly2、哈希与manifest均由源码生成，并通过运行时几何、关键场和检测终止门禁。
- 方案：正交加速飞行时间质谱，双级环栈反射镜，一级10环、二级5环。
- 标准离子：524 amu、+1电荷。
- 初始能量：均值5 eV、标准差0.4 eV，负能量重采样。
- 统一全局原点为检测器有效面中心与精确一阶时间聚焦面：`z=0`；`+z`从加速器指向反射器。
- 释放体：`1×1×1 mm³`，当前轴向范围`z=-18.92918680341103...-17.92918680341103 mm`。
- 分辨率：`R=m/FWHM_m`。窄峰时间域等价式为`R=T/(2*FWHM_t)`。
- `2.3548×sigma`只在高斯假设成立时作为FWHM代理；非高斯峰必须直接测量半高宽。
- SIMION常规统计N=5000；跨求解器严格比较优先使用同一固定ION粒子表。
- SIMION正式轨迹积分质量为8，由机器契约、同名Program加载段和IOB运行时门禁共同保证。
- SIMION稳定入口已冻结为0.05mm正式可移植交付与0.025mm轴向收敛参考；实现资产与SHA-256见
  `config/simion_stable_entry.json`。两者均为自包含目录且使用相同静态PA优先级；该清单不替代
  `baseline.json`的物理契约。

### 参数派生硬规则

本项目严格执行：`物理输入 → 公式 → baseline工程参数 → COMSOL/SIMION/CAD`。机器权威源是
`config/baseline.json`；任何软件现有文件、网格坐标、格式化结果或GUI显示均无权反向修改它。

反射器长度由`U0=2000 eV`、总无场长度`L=1200 mm`、一级长度`d1=120 mm`和二级余量系数
`1.0`按baseline中记录的公式派生。原始计算为`d2=86.8328157299974 mm`、
`L_refl=206.832815729997 mm`；工程长度统一保留小数点后4位，因此baseline值只能是
`d2=86.8328 mm`、`L_refl=206.8328 mm`。COMSOL曾因`%g`序列化得到`206.833 mm`，这是实现
舍入错误，不是新权威值；该错误转正已撤销并归档。长期门禁必须重新计算上述公式，同时检查
COMSOL写入精度、SIMION实体/Program以及CAD均服从同一工程值。

### 峰肩来源诊断

正式N=100同源粒子的z-to-TOF二次映射已经闭合到可解释状态。COMSOL日常档时间焦点顶点在
`z=1.58204 mm`，SIMION在`z=1.40316 mm`，相差`0.17888 mm`；二次曲率分别为
`0.00867780/0.00928120 us/mm²`，说明曲率接近而焦点位置发生平移。z单变量二次模型已分别解释
`93.22%/88.20%`的TOF方差，加入初始能量、x和y后升到`93.39%/88.66%`，所以右肩差异主要是
纵向时间聚焦映射，不是横向截断或能量表导入错误。

相同坐标的场采样进一步定位到加速段：释放区SIMION轴向Ez平均比COMSOL高`0.8396%`；反射器
内部（排除电极边界0.30 mm插值带）相对RMS差仅`0.000528%`。18、52、97号代表粒子在加速器
出口时SIMION比COMSOL晚`0.29--2.11 ns`，到反射器入口扩大为`16.57--22.84 ns`；反射后返回
入口时变为提前`12.64--14.88 ns`，最终检测面只残留`1.94--5.07 ns`。这证明两者的反射区传递
实质闭合，剩余峰肩由小的加速区场差经过时间聚焦放大并再次补偿形成。

### 2026-07-17 SIMION加速器轴向网格相位诊断

为检验非整数尺寸相对`dz=0.05 mm`网格的位置效应，保持全部机械坐标不变，把加速器几何在PA内
平移`0/0.0125/0.025/0.0375 mm`并反向补偿实例位置；四个相位均使用前后`0.2 mm`扩展裁边、同一
固定N=100和独立SIMION场。正式裁边与扩展裁边的平均TOF差仅`4.83e-7 ns`、FWHM差`2.83e-5%`，
排除裁边混淆。相位扫描的全加速段轴线场相对RMS最大仅`0.001718%`，释放区平均Ez跨度仅
`1.15e-7%`，不能解释COMSOL/SIMION现有`0.8396%`释放区平均场差。

粒子传递却对网格相位显著敏感：为可靠越过一格厚的透明数值栅网，全部相位统一使用`0.03 mm`
跳转缓冲时，平均TOF跨度`1.62898 ns`、同粒子TOF RMS差最大`2.46777 ns`；用`0.04 mm`复核时
平均TOF跨度`1.65565 ns`，两次跨度只差`1.64%`，约为当前跨求解器平均TOF差的`51--52%`。
直接FWHM跨度则从`32.89%`变为`19.35%`，说明峰宽仍与透明栅网跨越细节耦合。正式`0.005 mm`
缓冲下，半格相位粒子会在`grid1,z=3 mm`数值层splat；这不是机械传输失败，而是固定跳转距离未
越过该网格相位下的电极层。

结论：`surface=fractional`已使静电场基本不受相位影响，但当前“一网格点电极层+固定距离跳转”会
把网格相位放大为ns级逐粒子TOF差。非整数网格对齐是SIMION高精度粒子映射的重要误差源，但不是
现有平均场差的来源，也不能单独解释全部COMSOL/SIMION差异；不得通过改物理尺寸迁就网格。

同一固定N=100的0.5 mm收敛档为100/100命中、`R=28837.88`；正式1 mm日常档为
