# RF 四极杆项目状态

## 当前结论

`transport_no_collision + official_fixed_25.ion`已在COMSOL 6.4和SIMION 2020完成独立场、粒子推进、
统一事件表、manifest和GUI资产复验。它证明当前参考四极杆的RF-only径向约束与轴向传输，不证明碰撞
冷却、质量过滤、机械正式几何或整机连接。

面向集成的N=100接口候选在两个求解器中均100/100传输，但出口束斑、发散和均能未满足暂定相空间
一致性目标。因此项目状态是“官方回归有效、严格接口未闭合”。Static可执行；Candidate可产生可追溯
PASS/FAIL；Formal因未冻结机械几何和SolidWorks装配而有意阻断。

RF→oaTOF已完成最小真实侧孔、局部联合场、逐粒子按全局时钟进入、统一有限脉冲和稀疏状态输出的
N=100功能链。当前5 eV、100 amu、+1候选在`1.0×0.9 mm`孔上的最新清点为
`100输入 → 77外表面几何接受 → 40有限孔壁预测幸存 → 39脉冲前活动 → 39局部出口`；这只证明功能
贯通，不是S1资格PASS、优化传输率、分辨率闭合、紧凑储存或Formal整机连接。

## 资格与系统边界

- 参数链固定为`baseline + particle source + mode + interface → resolved → COMSOL/SIMION`。GEM、Lua、
  MATLAB脚本、MPH、PA和结果表均不得反写机器参数。
- `transport_no_collision`是不可变的N=25官方回归；`transport_interface_readiness`是在同一硬件上的
  N=100接口资格叠加，不是第二套四极杆。
- `mass_filter_reference`只通过求解器无关L0理论与电压语义检查，尚未运行质量扫描，不具备质量峰、
  分辨率或透过率资格。
- 碰撞模型尚未建立。旧150 mm碰撞脚本只保留拒绝执行短桩，不属于当前几何或物理合同。
- 传输四极杆和未来质量过滤四极杆应实例化同一硬件模板并分别绑定mode与位姿，不能复制两套近似几何。
- 当前S1局部联合模型不是完整仪器：不包含350 mm飞行管、反射器、探测器或机械连接器，不修改oaTOF
  Formal资产。

## RF→oaTOF当前候选

### 屏蔽、网格与局部场

参考RF几何在杆区没有连续接地侧壁。连续0 V圆柱壳候选只把内半径`19.776/26.368 mm`带入三维验证，
没有选定机械半径或壁厚。当前低成本三维混合网格为：杆工作核心`r≤8 mm`横向0.2 mm、外区1.0 mm、
轴向40层扫掠；两端四面体`hmax=0.5 mm`，`0.25 mm`只作参考。N=100配对诊断分类一致，且下游功能
裁决未因两级网格改变；不再授权0.125 mm局部加密。

S1局部模型采用连续屏蔽内部真空，不建立任意外部绝缘真空盒。它只包含RF内部真空与壳体、完整端板、
oa加速器内部真空和侧壁开孔。`1.0×0.9 mm`开孔相对闭孔对照的轴心Ez/电势相对RMS变化为
`5.5640e-4/1.7294e-5`；oa脉冲场向RF上游最大泄漏`0.001681 V/m`，实际RF峰值场在oa源中心附近
`5.853e-15 V/m`。这些结果授权N=100粒子功能候选，不等于数值收敛、分辨率或S1晋升。

### 入口孔与约束

当前oa耦合理论给出一级加速场160 V/mm和允许最高能量每电荷2080 V，空间相关z向完整宽度上限为
1.0 mm。候选取其90%，即z向完整孔高0.9 mm；横向孔宽当前取1.0 mm。孔径不是束流包络硬下限：
超出部分可以作为明确报告的几何损失，但进入资格验收前必须冻结最低通过率。

所有硬约束采用逻辑AND：同一尺寸的上限取全部适用上限的最小值，下限取全部适用下限的最大值；耦合
约束保留为二维可行域。理论、电场/分辨率、泄漏、高压、机械间距、装配公差和束流损失预算中的任何
必需项缺失，都不能得到S1 PASS。

### 连续束切片与脉冲

当前功能是连续四极杆束的时间切片弹射，不是积累或压缩。离子在入口面按各自全局到达时间进入oa
加速器的0 V预脉冲场，随后接受一个共享有限提取脉冲。定时器必须显式选择`mass_amu + charge_state`，
用真实逐粒子位置、速度和入口时间预测有限孔壁幸存者，再求其x质心到达oa理想源中心的共享时刻；混合
物种必须分组，禁止硬编码某个质量或能量的时间。

当前100 amu、+1、5 eV时刻为`t_pulse=35.831620768 µs`，脉宽1 µs。77个外表面接受者中37个在
4 mm入口壁终止，另1个在脉冲前撞排斥极；39个真实活动粒子的x质心为`−48.8153 mm`，相对目标
`−48.8 mm`误差`−0.0153 mm`，11个进入理想x区间、3个进入完整`1×1×1 mm³`参考体积，39个均通过
局部出口。活动束均能约5.0207 eV。

0.9 mm孔高只限制过孔位置，不消除z向速度。39个活动粒子入口z跨度0.3568 mm，
`vz=−217.8～+212.7 m/s`，漂移5.696～6.619 µs后脉冲前z跨度为2.8761 mm；直线速度外推与COMSOL
位置的z向RMS残差仅0.00249 mm。因此展宽主要是方向发散，不能把入口孔高误作下游束斑上限。

标准脉冲图必须在`x-z`和`x-y`两面同时显示参数化加速器、真实孔、理想源边界、活动离子、入口孔壁
损失和加速器内部损失。grid1/grid2使用各自baseline尺寸；大N标记自动缩放。粒子终止后冻结的有限坐标
只能作为损失位置，不能计入脉冲前活动束。

### 分阶段实现

| 阶段 | 最小物理增量 | 当前状态 |
|---|---|---|

| S1 共面开孔 | oa屏蔽开孔、共同0 V参考、局部联合场 | 39粒子完整功能链PASS；资格未闭合 |

“没有连接器”只允许表示S1没有独立连接器部件；仍必须真实切开oa屏蔽并求联合场。坐标变换只能注册
位姿，不能把粒子从RF出口平移到封闭oa内部。相对位移、有限间隙、变径、转向和聚焦均属于连接器的
真实物理输运。

### 状态、时间与场所有权

canonical交接量是每粒子一次的首次正向过面事件：身份、物种引用、全局时间、三维位置和三维速度。
动能、半径、发散、RF相位、年龄和求解器局部时间按需派生，不在接口重复保存密集轨迹或多份相同数据。

状态包保留`clock_epoch_id`、`instrument_time_us`、`lineage_age_us`、`particle_age_us`和
`last_component_elapsed_time_us`。全局时间从统一仪器触发起算；谱系年龄从根粒子出生累计；反应生成
子粒子时当前粒子年龄归零。静电消费者可以使用局部零时刻，但必须保留全局映射；时变场必须直接消费
全局时间或严格等价的相位偏移。

当前S1联合模型、canonical状态和SIMION workbench直接共用`oatof_global`，因此局部出口采用恒等交接，
不再维护数值相同的第二份`instrument_global`坐标或做无意义的来回变换。S2出现真实部件相对位姿后，
才分别把RF和oa位姿注册到`instrument_global`并唯一派生右手刚体变换；位置乘旋转并加平移，速度只乘
旋转，时间不变，非正交或行列式不为+1的旋转必须拒绝。接口附近重叠的RF和oa场归一个联合模型所有，
不能拼接两个封闭静电场或靠增加CSV列数替代场求解。
