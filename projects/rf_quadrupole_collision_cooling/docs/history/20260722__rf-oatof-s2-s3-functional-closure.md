# RF→oaTOF S2–S3功能闭环记录（2026-07-22）

DOC_STATUS: ARCHIVED_READ_ONLY

本文件冻结截至2026-07-22的RF四极杆到oaTOF连接实现过程。文中的“当前”“下一步”只按归档时点
解释；项目现状、机器参数和开放任务以[`../PROJECT.md`](../PROJECT.md)为准。S1之前的RF验证、网格
调查和早期集成过程见[`20260722_rf-validation-and-s1-integration.md`](20260722_rf-validation-and-s1-integration.md)。

## 里程碑与声明边界

连接任务从投影式数据直连推进到真实局部联合场、有限连接器、共享时钟脉冲和下游SIMION分析器，
最终形成一个累积S3入口。它证明以下功能在同一粒子状态链中可运行：

- RF出口首次正向过面事件进入参数化连接拓扑；
- COMSOL保持粒子ID、三维位置、三维速度、质量、电荷和全局时钟；
- 连续束粒子按各自到达时间进入0 V预脉冲加速器场；
- 所选物种共享一个有限1 µs提取脉冲；
- 局部加速器出口状态无投影续送至只读oaTOF SIMION分析器；
- SIMION保持原粒子ID、绝对出生时刻和同一脉冲结束时刻。

本里程碑不声明S2/S3资格PASS、最低传输率、数值收敛、峰形或分辨率、机械正式几何、CAD同步、
跨求解器场连续性或Formal整机连接。

## 累积阶段

| 阶段 | 新增的最小物理功能 | 归档结论 |
|---|---|---|
| S0 | 坐标、身份、时间和事件映射，无场输运 | 被后续阶段覆盖，只保留诊断 |
| S1 | 共面开孔、共同0 V参考和局部联合场 | 被后续阶段覆盖，只保留诊断 |
| S2 | 0或有限间距的被动连接器及无脉冲输运 | 0 mm与1 mm功能证据完成 |
| S3 | 共享时钟、有限脉冲和下游分析器 | N=100累积功能链完成 |

阶段是累积替代关系。归档时的生产入口为
`tests/cross_solver/run_s3_cumulative_chain.ps1`；S2、S3及下游单段runner作为该入口的内部阶段和独立
诊断工具，不再构成并行的当前工作流。

## 几何与拓扑

RF局部轴向面为：杆端`z=85.4 mm`、组件出口handoff面`z=90.2 mm`、独立传输检测面
`z=95.2 mm`。因此连接链在杆端下游4.8 mm取得物理状态，不使用下游9.8 mm处的独立检测面。

基础S2合同冻结同轴位姿、RF出口半径3.6 mm、接地圆柱连接腔和oa入口矩形孔。孔的完整尺寸为
`1.0×0.9 mm`；z向0.9 mm来自当前oa耦合理论1.0 mm完整宽度上限的90%设计值。孔只负责几何裁剪，
不把束流包络强行压缩到孔高。

`config/rf_to_oatof_connector_cases.json`从一份S2基础合同派生两个案例：

- `nominal_gap_1mm`：默认1 mm接地连接器；
- `direct_mating_gap_0mm`：共面连接，不创建独立连接器真空域。

负间距拒绝。两个案例使用同一孔径、位姿、场构建器、粒子合同和累积入口，不能作为改变了多个变量的
性能比较。

## 粒子接口与数值重启

canonical接口只保存每粒子一次的物理过面事件：身份、物种引用、frame、clock epoch、全局时间、
三维位置和三维速度。动能、半径、发散、RF相位和局部历时按需派生；接口不复制密集轨迹。

0 mm案例暴露了COMSOL不能把孔外粒子释放到端壁实体中的问题。最终实现先在精确物理接口面按孔径
分类：孔外粒子立即记录为入口壁损失，只有孔内粒子进入CPT。孔内粒子暂以`0.001 mm`下游数值重启，
并按法向速度同步推进启动时刻、按三维速度推进位置。该值不是器件间距、孔厚或机械公差，也不参与
通过率判定。未来应采用COMSOL原生接口面或连续真空内部边界初始化并删除该偏移。

## 共享时钟与脉冲

本功能是连续束时间切片，不是积累、压缩或捕获阱。每个离子按自己的`instrument_time_us`进入加速器
0 V预脉冲场；定时器显式选择`mass_amu + charge_state`，用真实入口位置、速度和时刻预测有限孔幸存者，
再求其x质心到达oa理想源中心的共享脉冲时刻。混合物种必须分组，禁止硬编码某个质量或能量的时间。

默认1 mm案例的脉冲起点为`36.112152843 µs`；0 mm案例为`35.831620768 µs`；脉宽均为1 µs。
脉冲前z向展宽来自入口`vz`发散和不同漂移时间，0.9 mm孔高不是下游z跨度上限。

## 被S3取代的S1证据

S1质心定时运行曾把39个真实局部出口状态通过
`20260722_100000__sim__cross__rf-oatof-s1-centroid-pulse-downstream__n39__r04`送入oaTOF分析器，得到
39/39下游命中。粒子在`36.454527～36.486870 µs`进入下游时仍处于同一
`35.831621～36.831621 µs`脉冲，未重启波形。探测落点质心为`(-1.3103,-0.1267) mm`，RMS半径
`5.6711 mm`，最大半径`11.6858 mm`；N=39诊断给出TOF FWHM `21.839 ns`、质量FWHM
`0.14009 Da`和`R≈713.8`，但小样本及双峰形不构成分辨率资格。

analysis-only运行`20260722_111500__analysis__python__rf-oatof-s1-state-chain-audit__n39`验证COMSOL局部
出口到canonical的位置/速度残差为0，ION反解后的全局速度最大相对残差`2.02e-14`，SIMION初始位置和
能量残差为0，链年龄最大残差`7.11e-15 µs`，探测全局时间残差为0。

随后发现r04的canonical入口表沿用旧投影面`x=-62.8 mm`，而真实物理入口为`x=-67.8 mm`。运行
`20260722_120000__analysis__python__rf-oatof-s1-entry-boundary-repair__n100__r02`从原始RF出口重建物理面
状态；与旧表相比只有x统一改变`-5.0 mm`，身份、时间、其他位置分量、三维速度和能量均不变。该修复
建立了后续S2/S3求解器启动前的物理面与frame校验。

## S2构建与无脉冲证据

1 mm几何运行`20260722_140611__build__comsol__rf-oatof-s2-passive-connector__gap1`得到21个几何域，
只验证连接腔、oa入口孔和共享构建器，不建网格、不求场、不放粒子。场运行
`20260722_142733__analysis__comsol__rf-oatof-s2-no-pulse-field__gap1__r03`建立200712个网格单元并求得
oa静态场与RF 100 V单位场；5个合同探针均有限，杆区1 mm离轴探针单位场模为`12534.6156 V/m`。

随后N=100运行`20260722_150059__sim__comsol__rf-oatof-s2-passive-connector__n100__r02`得到61个孔内
oa入口事件和39个连接器/入口壁损失，并验证ID、frame、clock epoch、年龄、质量/电荷及速度—能量
一致性。它被最终统一入口的1 mm S2运行取代，但计数保持一致。

初始化缺陷运行`20260722_135659__build__comsol__rf-oatof-s2-passive-connector__gap1`、首次场选择错误、
粒子链Python启动遗漏及MATLAB布尔序列化问题均以失败三件套保留。它们只证明异常收尾和根因，不是
几何或物理证据。

## 关闭的实现问题

- 从RF出口投影到oa理想源的早期方案被真实局部场输运取代。
- 旧handoff表把物理入口写成`x=-62.8 mm`，而实际释放面为`x=-67.8 mm`；修复后物理面、定时器和
  MATLAB入口均在求解器启动前交叉校验。
- S2首次build、场选择、Python启动和MATLAB布尔序列化问题均以失败三件套保留在artifacts，最终入口
  不再依赖第二套Python解析或共享目录猜测来源。
- 0 mm首次粒子运行把所有粒子偏移进下游，孔外粒子进入实体后没有有限状态；物理面预分类修复了该
  错误。一次命令层短超时中断也形成合法`interrupted`三件套。
- 单独的S2、S3和SIMION命令链被一个累积入口取代；显式source run和冻结合同建立唯一依赖链。
